# Home Server Hardware

## Specs

- HPE ProLiant MicroServer Gen11 (P68821-421)
- 4x GBit LAN
- 2x USB3.2 Gen2
- 4x USB3.2 Gen1
- VGA + DisplayPort
- 1 PCIe 5 x16
- 1 PCIe 4 x8
- IPMI with ILO Module P65741-B21 incl. M.2 Slot
  - Key M
  - 2280 or 22110 or
- Xeon E-2434
- 1x 16GB DDR5 4400 (4 Slots in total)
- 180W external Powersupply
